ng-FitTextDynamic makes font-sizes flexible. Use it whenever you're displaying dynamic text of different and unknown widths, or even for static text (though you might consider using ng-fittext if you only need to resize static text).

{{dynamicText}}

In addition to resizing on window events, you can also resize whenever you want (like when something finishes loading). Just broadcast the even given in dynamic-text, and make sure you have min and max values set!

Install and Inclusion

Grab it with Bower:

bower install ngFitTextDynamic

Include it in your AngularJS application

var myApp = angular.module( 'myApp', [ 'ngFitTextDynamic' ] );

Apply it to your text

<h1 dynamic-text>FitText</h1>

There are also additional attributes which can be used.

Specifying a value for dynamic-text allows you to set an event on which to trigger text resizing. dynamic-text-min and dynamic-text-max allow you to set upper and lower limits. This can also be set globally (applying to all uses) via the dynamicTextConfigProvider as seen below.

<h1 dynamic-text="event name" dynamic-text-min="12" dynamic-text-max="50">ngFitText</h1>

Debounce

Debouce (limiting the rate at which DynamicFitText will be called on window resize) is available. You can pass initialization parameters to the ngFitText constructor to achieve this:

myApp.config( function( dynamicTextConfigProvider ) { dynamicTextConfigProvider.config = { debounce: true, delay: 1000, min: 20, //minimum global font size max: 100, //maximum global font size }; // OR dynamicTextConfigProvider.config.debounce = true; dynamicTextConfigProvider.config.delay = 1000; dynamicTextConfigProvider.config.min = 20; dynamicTextConfigProvider.config.max = 100; });
